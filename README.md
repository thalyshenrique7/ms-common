# MS Common

Módulo compartilhado contendo eventos e DTOs utilizados pelos microservices **order-service** e **stock-service**

Este projeto contém apenas classes de contrato (eventos/enums), sem regras de negócio ou configurações.

## Como usar

Instale localmente:

mvn clean install


Adicione como dependência:

```xml
<dependency>
    <groupId>com.ms</groupId>
    <artifactId>common</artifactId>
    <version>0.0.1-SNAPSHOT</version>
</dependency>
```
